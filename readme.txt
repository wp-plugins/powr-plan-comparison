    == Description ==
    POWr Plan Comparison is a mobile-responsive, fully-customizable plugin that you can edit **right in the page**.
    Add the widget to your site, or use the shortcode **[powr-plan-comparison]** to add Plan Comparison to any post.
    <br><br>
    Feature your plans, products, features, and more. POWr Plan Comparison allows you to create beautiful, mobile-responsive comparisons in seconds. <br><br>Choose from professionally designed templates, or fully customize to fit your brand. Create an unlimited number of plans to compare, add as many features as you'd like, and customize all fonts, colors and backgrounds. You can even select a style a "highlighted" plan to improve your user engagement. <br>
    <br><br>
    The basic version of the POWr Plan Comparison is free!
    <br><br>
    POWr plugins can be used an virtually any website, learn more at [www.POWr.io](http://www.powr.io/)
    <br><br>
    Features:
    <br><br>
    * Compare an unlimited number of plans<br> * Add unlimited number of features<br> * Customize all fonts, colors, and background to fit your brand<br> * Customizable "highlighted plan" for emphasis<br> * Language support - customize text to use any language<br> * Mobile responsive - designed to look great on desktop, tablets, and mobile


    == Installation ==
    To install POWr Plan Comparison
    <br><br>
    1. Upload 'powr-plan-comparison.php' to the '/wp-content/plugins/' directory
    <br>
    2. Activate the plugin through the 'Plugins' menu in WordPress
    <br><br>
    To add POWr Plan Comparison as a widget, go to **Appearance->Widgets**, and drag POWr Plan Comparison to where you'd like it to appear.
    <br><br>
    To add Plan Comparison within a post or a page, simply write the shortcode **[powr-plan-comparison label='MY LABEL']** (including the brackets) where you'd like your Plan Comparison to appear. Then visit your page!
    <br><br>
    Want a more detailed tutorial? Visit [How to add Plan Comparison to your Wordpress site](http://www.powr.io/tutorials/how-to-add-plan-comparison-plugin-to-your-wordpress-site)


    == Frequently Asked Questions ==
    = How can I add Plan Comparison to a blog post or page? = Just add the shortcode **[powr-plan-comparison label='MY LABEL']** (including the brackets) in your post where you'd like your Plan Comparison to appear. Then visit your page!

    = How can I add  Plan Comparison to my site multiple times? = Just be sure to specify different label text, eg **[powr-plan-comparison label='MY LABEL 1']**, **[powr-plan-comparison label='MY LABEL 2']** for each time you use the plugin

    = Why do you require me to sign up? = Signing up is what allows us to track which plugins are yours, and allows you to edit them right in your page.

    = Will my users see the settings icon in the corner of the Plan Comparison? = No! You only see the settings icon because you own the plugins. To temporarily hide it, click on your page and then type 'p + down arrow' (the letter p followed by the down arrow). You can show the icon again by typing 'p + up-arrow'.

    = I can't see the settings icon anymore, what happened? = You are just logged out of POWr.io. You can show the icon again by typing 'p + up-arrow'.

    = Is POWr Plan Comparison Free? = Yes, the basic version of Plan Comparison is completely free!

    = Why should I upgrade? = Upgrading removes all POWr watermarks, speeds up widget loading speed, gets you premium support, and most importantly supports the creation of more great plugins!
